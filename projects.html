Implementation of ML Algorithms from Scratch

Music Streaming Service
Mentors Saravanan, Aaris and Anisha.
Streaming music, or more accurately streaming audio, is a way of delivering sound — including music — without requiring you to download files from the internet. In this project, we will be building a simple music streaming application with user profiles that plays music stored in a server.
Internet Radio
Mentors Akshay and Anisha.
An internet radio receiving service that can fetch radio streams from all over the world and play the streams at the user’s convenience. A user can play his favourite radio stations by filtering the streams through location, genre and language.
Web Crawler
Mentors Chaitanya and Yasaswi Raj.
A web crawler downloads and indexes content from all over the Internet. The goal of such a bot is to learn what (almost) every webpage on the web is about, so that the information can be retrieved when it is needed.
HTTP Server
Mentors Sai Kiran and Yasaswi Raj.
An HTTP server is software that understands URLs (web addresses) and HTTP (the protocol your browser uses to view webpages). An HTTP server can be accessed through the domain names of the websites it stores, and it delivers the content of these hosted websites to the end user's device. In this project, we will be building an HTTP server from scratch in C.
Password Keeper
Mentors Poojitha and Aaris.
A password manager is a computer program that allows users to store and manage their personal passwords for online services. It will also assist a user in generating and retrieving complex passwords, potentially storing such passwords in an encrypted database or calculating them on demand.

In this project, we will be building a text-based and a graphic based interface of the Chess game entirely from scratch using Python. It will be a PvP game i.e. A player competes against another player in each game.
